You have N files.
Each file contains the transcript of records of one student in the form
SubjectName Mark Credits

The file is named Surname.txt

for each student you have to show (i.e., print on the screen):
Surname.txt WeightedAverage MaxVote OldestDate-MostRecentDate

Write a sequential program first, where each file is carried out by a child process.
In this case, the list of filenames is provided in an input file that can be provided as an argument.
